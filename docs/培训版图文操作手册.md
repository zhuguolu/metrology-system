版本：V1.0  
适用对象：普通用户、管理员、外链访客

## 1. 手册说明

本手册用于培训新用户快速掌握系统的常用操作。内容采用“页面说明 + 操作步骤 + 培训提示”的写法，适合：

- 新员工上岗培训
- 部门内部功能宣讲
- 管理员交接
- 打印后作为纸质操作指南

建议在正式发放前，补充各页面真实截图。本文已经按章节预留了“截图建议”，后续只需要把对应页面截图插入即可。

## 2. 系统角色说明

系统内常见角色分为三类：

### 2.1 普通用户

普通用户可按授权使用以下功能：

- 查看设备台账
- 记录校准
- 查看待办
- 使用文件中心
- 访问 WebDAV 挂载
- 提交需要审核的数据变更

### 2.2 管理员

管理员默认拥有全部功能，同时可执行：

- 用户创建与权限分配
- 部门管理
- 使用状态维护
- 审核变更申请
- 查看审核历史
- 调整系统参数
- 设置外链分享后缀

### 2.3 外链访客

外链访客无需登录，可通过分享链接访问公开目录。根据分享设置，访客可：

- 浏览文件夹
- 搜索当前目录内容
- 在线预览图片、PDF、文本、音视频、Office 文档
- 下载文件
- 输入访问密码后继续浏览

截图建议：系统登录页、主界面左侧菜单、公开分享页首页

## 3. 登录与首页认识

### 3.1 登录系统

操作步骤：

1. 打开系统地址。
2. 输入用户名。
3. 输入密码。
4. 点击“登录”。
5. 登录成功后进入系统首页。

培训提示：

- 用户名或密码错误时，页面会直接提示。
- 若管理员修改了用户名，通常需要重新登录。
- 忘记密码时，请联系管理员重置。

截图建议：登录页面，标注“用户名”“密码”“登录按钮”

### 3.2 首页主结构

登录后页面通常分为三部分：

- 左侧导航栏：切换模块
- 顶部栏：显示当前页面名称、日期、刷新按钮、工作台入口
- 主内容区：执行具体业务操作

常见菜单包括：

- 总览看板
- 设备台账
- 校准管理
- 我的待办
- 我的文件
- 数据审核
- 使用状态
- 部门管理
- 变更记录
- 网络挂载
- 用户管理
- 系统设置

培训提示：

- 手机端可通过顶部菜单按钮展开导航。
- 顶部“工作台”适合快速进入常用业务模块。
- 顶部“刷新”会刷新当前页面缓存，不等于退出登录。

截图建议：主界面整体布局图，标注左侧导航、顶部栏、内容区

## 4. 总览看板

总览看板用于查看系统总体运行情况，适合管理者和日常值班人员快速了解设备状态。

页面可查看：

- 设备总数
- 本月待校准数量
- 有效设备数量
- 失效和即将过期数量
- 近 6 个月校准趋势
- 设备有效性分布
- 部门设备统计

常见用途：

- 开会前快速查看总体情况
- 每周检查过期风险
- 了解各部门设备有效率

培训提示：

- 看板主要用于查看，不直接修改数据。
- 若统计结果异常，先检查设备台账和校准记录是否完整。

截图建议：看板首页，重点标注四个统计卡片和图表区域

## 5. 设备台账

设备台账是系统核心页面，用于新增、维护、查询和导出设备资料。

### 5.1 查询设备

支持以下筛选条件：

- 关键字搜索
- 部门
- 有效状态
- 使用状态

操作步骤：

1. 在搜索框中输入设备名称、计量编号、责任人、资产编号或出厂编号。
2. 结合部门、有效状态、使用状态筛选。
3. 查看查询结果与分页信息。
4. 必要时点击“重置”恢复默认列表。

截图建议：设备台账页面顶部筛选栏

### 5.2 新增设备

点击“新增设备”后，表单通常分为四部分：

- 基本信息
- 采购与技术
- 校准信息
- 附件与备注

操作步骤：

1. 点击“新增设备”。
2. 在“基本信息”中填写仪器名称、计量编号、部门、责任人、位置、使用状态等。
3. 在“采购与技术”中填写采购日期、采购价格、技术参数。
4. 在“校准信息”中填写检定周期、本次校准日期、校准结果。
5. 在“附件与备注”中上传设备图片、校准证书，填写备注。
6. 点击“保存”完成新增。

培训提示：

- 计量编号建议唯一，便于后续查询和审核。
- 检定周期和校准时间会影响“下次校准”和“有效状态”的自动计算。
- 设备图片支持上传两张，适合一张正面照、一张铭牌照。

截图建议：新增设备弹窗四个页签

### 5.3 编辑设备

操作步骤：

1. 在设备列表中找到目标设备。
2. 点击“编辑”。
3. 修改需要更新的字段。
4. 点击“保存”。

培训提示：

- 若系统启用了审核流程，普通用户的修改可能先进入审核，不会立即生效。
- 管理员审批通过后，修改内容才会正式落库。

### 5.4 预览设备

点击设备名称或“预览”按钮，可查看：

- 基本资料
- 计量编号
- 部门与责任人
- 上次校准与下次校准
- 校准结果
- 备注
- 图片和证书

适合在不进入编辑状态时快速核对信息。

### 5.5 导入、导出与模板

页面支持：

- 下载导入模板
- Excel 批量导入
- 导出当前筛选结果
- 导出全部设备数据

培训提示：

- 导入前务必先下载模板，避免字段顺序不一致。
- 大批量导入后，应抽查几台设备是否显示正常。

### 5.6 批量操作

勾选设备后可执行：

- 全选当前页
- 清空选择
- 批量改部门
- 批量改责任人
- 批量改位置
- 批量删除

培训提示：

- 批量删除风险较高，建议操作前先导出备份。
- 若属于受审核场景，删除可能进入审核流程。

截图建议：列表页、多选条、批量操作区

## 6. 校准管理

校准管理用于集中维护设备的校准周期、校准时间和结果。

### 6.1 查询校准对象

支持以下筛选：

- 名称、编号、责任人搜索
- 使用部门
- 有效性
- 使用状态
- 下次校准日期区间

### 6.2 记录单台设备校准

操作步骤：

1. 找到目标设备。
2. 点击“记录校准”。
3. 填写本次校准时间。
4. 选择检定周期。
5. 选择校准结果。
6. 填写备注。
7. 点击“保存校准记录”。

系统会自动更新：

- 上次校准时间
- 下次校准时间
- 有效状态

### 6.3 批量记录校准

操作步骤：

1. 勾选多台设备。
2. 点击“批量记录校准”。
3. 输入统一的校准信息。
4. 保存后批量更新。

适用场景：

- 同一批设备集中送检后统一回写
- 周期一致的设备批量维护

### 6.4 手机端快速编辑

移动端支持“快改”功能，可快速修改：

- 使用部门
- 责任人
- 位置
- 使用状态
- 检定周期
- 本次校准时间
- 校准结果
- 备注

截图建议：校准管理列表、记录校准弹窗、手机端快改弹窗

## 7. 我的待办

“我的待办”聚焦校准即将过期或已经失效的设备，方便责任人员集中处理。

### 7.1 待办页面能看什么

页面通常会显示：

- 已失效数量
- 即将过期数量
- 待处理总数
- 涉及部门数

列表中可看到：

- 紧急程度
- 仪器名称
- 计量编号
- 使用部门
- 责任人
- 下次校准日期
- 逾期天数

### 7.2 处理待办

操作步骤：

1. 使用搜索和筛选定位目标设备。
2. 点击设备名称查看详情。
3. 点击“记录校准”处理单台设备。
4. 或勾选多条记录后进行“批量记录校准”。

培训提示：

- 红色通常表示已失效，优先处理。
- 黄色通常表示即将过期，可提前安排送检。

截图建议：待办页统计卡片、列表、记录校准按钮

---

## 8. 我的文件

“我的文件”用于存放与设备、校准、资料管理相关的各类文件。

### 8.1 页面功能概览

支持以下常用操作：

- 按目录浏览文件
- 搜索当前文件夹
- 上传文件
- 新建文件夹
- 扫描同步
- 图标视图和列表视图切换
- 文件预览
- 文件下载
- 重命名
- 移动
- 删除
- 文件夹外链分享

### 8.2 新建文件夹

操作步骤：

1. 进入目标目录。
2. 点击“新建文件夹”。
3. 输入文件夹名称。
4. 点击“创建”。

### 8.3 上传文件

支持两种方式：

- 点击“上传文件”选择本地文件
- 直接把文件拖拽到页面中上传

页面会显示上传进度，包括：

- 单文件进度
- 批量总进度
- 完成状态

培训提示：

- 上传时请先确认自己在正确目录下。
- 页面会提示“当前目录”为哪一个文件夹。

### 8.4 搜索当前文件夹

页面顶部搜索框只搜索当前打开的文件夹，不会跨整个文件库全局搜索。

适合场景：

- 当前目录文件较多时快速定位
- 明确知道文件就在当前层级

### 8.5 文件操作

文件或文件夹常见操作包括：

- 双击文件夹进入
- 双击文件预览
- 右键打开快捷菜单
- 下载文件
- 移动到其他文件夹
- 重命名
- 删除

### 8.6 多选操作

点击“多选”后，可批量选择多个项目，并执行：

- 清空选择
- 删除已选

### 8.7 只读授权目录

若某个目录来自管理员只读授权，页面会提示：

- 当前目录为只读授权目录
- 仅支持查看和下载

此时通常不能：

- 上传
- 重命名
- 删除
- 移动
- 新建文件夹

截图建议：文件首页、上传进度、右键菜单、只读提示、多选栏

## 9. 文件夹外链分享

系统支持把文件夹生成公开分享链接，供外部人员访问。

### 9.1 创建分享

操作步骤：

1. 在“我的文件”中找到目标文件夹。
2. 右键该文件夹。
3. 点击“外链分享”。
4. 在弹窗中设置分享参数。

可配置项包括：

- 是否允许下载
- 失效时间
- 是否启用访问密码
- 访问密码内容
- 分享链接

管理员额外可设置：

- 链接后缀

例如：

- `/share/manuals_2026`
- `/share/device_docs`

培训提示：

- 链接后缀仅管理员可设置。
- 后缀只支持字母、数字、中划线、下划线。
- 若后缀已被占用，系统会提示更换。

### 9.2 关闭分享

在分享弹窗中点击“关闭分享”，即可使原链接失效。

截图建议：分享弹窗，标注“允许下载”“失效时间”“访问密码”“链接后缀”

## 10. 公开分享页使用说明

公开分享页已经优化为“文件预览门户”布局，适合对外发资料。

### 10.1 访客可执行的操作

访客打开链接后，可：

- 浏览当前目录
- 点击左侧目录项
- 搜索当前目录内的文件和文件夹
- 在线预览支持的格式
- 进入子文件夹
- 返回根目录
- 下载文件

### 10.2 支持在线预览的文件类型

当前公开分享页支持以下预览：

- 图片
- PDF
- 文本
- 音频
- 视频
- Office 文档

说明：

- Office 预览依赖在线预览服务。
- 在本地 `localhost` 或 `127.0.0.1` 环境下，Office 在线预览通常不可用。
- 加密分享场景下，Office 在线预览可能受外部服务访问限制影响。

### 10.3 加密分享访问

若分享启用了访问密码：

1. 打开分享链接。
2. 输入访问密码。
3. 点击进入分享。
4. 通过验证后继续浏览和预览。

截图建议：公开分享首页、输入密码页、左侧目录搜索、右侧预览面板

## 11. 数据审核

数据审核用于控制设备资料变更的审批过程，尤其适用于对台账准确性要求较高的单位。

### 11.1 管理员待审核

管理员可在“待审核”页查看：

- 提交人
- 操作类型
- 设备名称
- 提交时间
- 备注

可执行操作：

- 查看详情
- 审批通过
- 驳回

### 11.2 我的申请

普通用户可查看自己提交的申请，状态包括：

- 待审批
- 已通过
- 已驳回

若被驳回，可查看驳回原因后再次修改提交。

### 11.3 审核详情

详情页可查看：

- 申请流程
- 原始值与修改后值对比
- 备注
- 驳回原因

培训提示：

- 审核通过后，变更正式生效。
- 审核驳回后，原数据保持不变。

截图建议：待审核列表、详情对比页、审批通过/驳回弹窗

- 操作类型
- 处理状态

- 操作类型
- 提交人
- 处理状态

## 13. 用户管理

用户管理主要供管理员使用。

### 13.1 创建用户

操作步骤：

1. 进入“用户管理”。
2. 点击“创建用户”。
3. 输入用户名和密码。
4. 选择账号角色。
5. 若为普通用户，继续分配部门、功能权限、只读文件夹授权。
6. 点击“创建用户”。

### 13.2 分配功能权限

常见权限包括：

- 设备查看
- 设备新增
- 设备编辑
- 设备删除
- 校准记录
- 使用状态管理
- 用户管理
- 文件访问
- WebDAV 访问

培训提示：

- 管理员拥有全部权限。
- 普通用户可按岗位最小授权原则分配。

### 13.3 只读文件夹授权

管理员可给普通用户分配若干只读目录。授权后：

- 用户可以进入并浏览这些目录
- 用户可以下载文件
- 用户通常不能修改目录内容

### 13.4 重置密码

管理员可直接为普通用户设置新密码，保存后立即生效。

截图建议：创建用户弹窗、权限设置页面、只读文件夹授权下拉框

## 16. 系统设置

系统设置分为两部分。

### 16.1 个人账号安全

用户可在此页面：

- 修改用户名
- 修改密码

操作提醒：

- 修改用户名后通常需要重新登录。
- 修改密码时必须输入当前密码。

### 16.2 校准参数设置

仅管理员可设置：

- 即将过期预警天数
- 失效判定天数

这些参数会影响设备有效状态的判断逻辑。

示例：

- 距离失效日期少于预警天数时，显示“即将过期”
- 超过失效判定阈值后，显示“失效”

截图建议：系统设置页，标注账号安全和校准参数区块

## 17. 网络挂载 WebDAV

系统支持通过 WebDAV 连接外部网盘或文件服务。

### 17.1 可做什么

- 添加挂载点
- 编辑挂载点
- 删除挂载点
- 测试连接
- 浏览远程目录
- 上传文件到远程目录
- 下载远程文件

### 17.2 新增挂载点

操作步骤：

1. 点击“添加挂载”。
2. 输入名称。
3. 输入 WebDAV 地址。
4. 输入用户名和密码。
5. 先点击“测试连接”。
6. 测试成功后点击“添加”。

适配场景：

- 坚果云
- Nextcloud
- 其他支持 WebDAV 的网盘或文件服务器

## 18. 培训常见问题

### 18.1 为什么我看不到“我的文件”或“网络挂载”菜单？

可能原因：

- 当前账号没有对应功能权限
- 账号不是管理员
- 文件功能未授权

### 18.2 为什么我能看到文件，但不能上传或删除？

可能原因：

- 当前目录是管理员分配的只读文件夹
- 仅允许查看和下载，不允许修改

### 18.3 为什么我修改了设备信息，没有立刻生效？

可能原因：

- 当前系统启用了审核流程
- 修改内容已经进入待审核队列

### 18.4 为什么外链可以打开但不能下载？

可能原因：

- 分享创建时未勾选“允许下载”
- 当前分享只允许浏览和预览

### 18.5 为什么 Office 文档在分享页打不开？

可能原因：

- 当前是本地地址访问
- 外部在线预览服务无法访问你的文件地址
- 当前分享启用了密码，外部服务无法带密码访问
